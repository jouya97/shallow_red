# CPU baseline smoke evaluation — 2026-07-21

This report records the first Phase 1 evaluation of the rules-verified match
harness. It is a directional smoke test, not a release claim.

## Setup

- Rules and adjudication: `python-chess` 1.11.2
- Opponent: Stockfish 18 installed from Homebrew
- Opponent budget: 1,000 nodes per move, one thread, 16 MB hash
- Positions: 100 unique deterministic positions generated by six uniformly
  sampled legal plies from the standard initial position
- Pairing: each target played every position as White and Black
- Games per target: 200
- Draw claims: disabled; automatic orthodox draws remain active
- Maximum game length: 300 plies; truncations are not counted as draws
- Seed: 20260721

Commands:

```bash
uv run worst-chess smoke \
  --target random --pairs 100 --openings 100 --opening-plies 6 \
  --opponent-nodes 1000 --max-plies 300 \
  --output artifacts/gate2-random

uv run worst-chess smoke \
  --target heuristic --pairs 100 --openings 100 --opening-plies 6 \
  --opponent-nodes 1000 --max-plies 300 \
  --output artifacts/gate2-heuristic
```

## Results

| Target | Self-checkmate | Draw | Accidental win | Protocol failure | Median plies | Mean plies |
|---|---:|---:|---:|---:|---:|---:|
| Uniform legal random | 200/200 (100%) | 0 | 0 | 0 | 33.5 | 34.04 |
| Sacrifice/king-exposure heuristic | 200/200 (100%) | 0 | 0 | 0 | 29.0 | 29.79 |

Both policies reached the primary outcome in every game, so the lexicographic
speed tie-break is informative in this suite. In matched position/color pairs,
the heuristic reached its own checkmate **4.25 plies sooner on average** than
random. A 10,000-resample paired percentile bootstrap gave a 95% interval of
**[1.52, 6.88] plies sooner**. The heuristic was faster in 115 pairs, tied in
9, and slower in 76.

Color-specific mean plies:

| Target | As White | As Black |
|---|---:|---:|
| Uniform legal random | 34.82 | 33.26 |
| Sacrifice/king-exposure heuristic | 27.94 | 31.64 |

## Interpretation

The smoke gate passes:

- 1,000,015 legal move/action transitions across 45,375 positions passed
  collision, mask, and encode/decode verification (88,361 transitions/second);
- 400 total games completed with no illegal moves or protocol failures;
- every game artifact contains auditable FEN, legal action, move, and PGN data;
- color pairing and deterministic opening generation reproduced cleanly;
- the explicit losing heuristic improves time-to-self-checkmate over random on
  the matched suite.

The 100% self-checkmate ceiling is expected against a conventional engine that
actively tries to win. It means future ordinary-opponent comparisons need
stronger discrimination from time-to-mate, weaker/human-like opponent strata,
and resistant opponents. This result does not measure adversarial selfmate
ability and is not evidence that the game is solved.

The raw reports and PGNs are generated under `artifacts/` and intentionally
gitignored because later suites will be substantially larger.
